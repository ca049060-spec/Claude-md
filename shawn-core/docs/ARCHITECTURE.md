# ShawnOS Architecture

## Purpose
Convert Shawn's accumulated AI work into a practical operating system.

## Layers
1. Command: what matters now
2. Context: user profile, current projects, constraints
3. Source: Drive, Gmail, Calendar, local files
4. Model: legal, medical, business, investment, family, joy
5. Tool: skills, scripts, dashboards
6. Ledger: commitments and external proof
7. Audit: Witness/Vale reviews
8. Joy: non-instrumental life actions

## Main package
`shawn-core` supplies skills and agents. It does not store private source data.

## Principle
Build the smallest thing that changes the outside world.
