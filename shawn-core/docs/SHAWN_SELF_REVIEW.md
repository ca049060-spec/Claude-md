# Shawn Self-Review Sandbox

This repository includes a built-in review persona: Shawn-as-reviewer.

## Review questions
1. Does this reduce cognitive load?
2. Does this protect legal/medical/money/family risks?
3. Does it create an external-world action?
4. Does it avoid storing private source data?
5. Does it stop overbuilding?
6. Would I actually use this tomorrow?

## Pass condition
The repo is useful only if at least one skill produces a real-world change within 7 days.

## Initial verdict
Strong foundation. Not perfect until installed and used once.
