# vacs-score-name

## Purpose
Gate investment ideas with VACS+.

## Workflow
1. Pre-signal triage: 7 flags.
2. If 3+ flags are red, reject and stop.
3. Full VACS scoring only for survivors.
4. Entry score caps actionable rating.
5. Log result in holdings/rejections/watchlist.
6. Define kill-shot field.

## Hard vetoes
- going-concern language
- dilution >10%/yr
- insider cluster-selling
- pre-revenue with no credible <24-month path
- no primary-source catalyst

## Rule
After a clear verdict, the next step is action or no action, not another research spiral.
