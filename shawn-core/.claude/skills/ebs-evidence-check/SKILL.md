# ebs-evidence-check

## Purpose
Prevent overstatement in the EBS legal file.

## Reliability labels
- GREEN: verified primary source
- PLAIN: corroborated across sources
- RED: cuts against case / needs caution
- EXCLUDED: unreliable or out of scope

## Use when
- adding a claim to EBS record
- preparing counsel package
- reading transcript/financial workbook
- comparing versions

## Procedure
1. State the claim.
2. Identify source file(s).
3. Label reliability.
4. Note what would disprove it.
5. Add open item if source is missing.
6. Never rely on excluded rental-modified records.

## Output
- Claim
- Evidence
- Reliability
- Risk
- Counsel open item
