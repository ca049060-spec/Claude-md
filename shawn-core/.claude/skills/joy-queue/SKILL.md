# joy-queue

## Purpose
Counter doom/defense mode with low-friction joy.

## Rules
- Must be doable in under 2 hours.
- Must not require a new system.
- Must include at least one physical-world option.
- Prefer father-son, County, creative, tactile, or food/water/fire/movie activities.

## Output
1. low-energy option
2. father-son option
3. creative option
4. outside option
5. one ridiculous thing

## Ban
No productivity masking as joy.
