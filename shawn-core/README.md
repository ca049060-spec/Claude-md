# shawn-core

A private Claude Code / ShawnOS plugin repository.

This repository converts Shawn's accumulated Claude/NIKA/North work into a reusable, installable operating system for practical work:

- truth-first AI collaboration
- legal/MTO evidence organization
- EBS evidence discipline
- Seals Go Pro launch execution
- VACS+ investing discipline
- commitment tracking
- anti-overbuilding safeguards
- joy / creative life prompts

This repo intentionally contains **no private source evidence**. It contains reusable skills, schemas, agents, templates, and governance.

## Fast start

```bash
python scripts/validate_repo.py
python scripts/install_local.py --dry-run
```

Then copy or install:

- `.claude/skills/*` into your Claude Code skill directory
- `.claude/agents/*` into your Claude Code agent directory
- `CLAUDE.md` into the project root you want Claude Code to load

## The core rule

Nothing becomes a durable system unless it produces real-world change.

The point is not to build a cathedral. The point is to make the phone call, submit the form, book the meeting, send the draft, update the ledger, collect the review, or choose the trade.

## Privacy

Do not commit raw medical, legal, family, Gmail, financial, or evidence exports here.

Use source manifests and citations, not full private records.
